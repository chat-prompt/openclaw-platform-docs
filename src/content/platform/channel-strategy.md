# 02. 축1: 플랫폼 전략

> 최종 수정: 2026-03-25

## 채널 구조 — 3단계 진화

### 1단계: 구경꾼 — 카톡 오카방

| 항목 | 상세 |
|------|------|
| 현재 | 231명, 주간 1,143건 |
| 목적 | 사람 많이 모으기. 관심 형성 + 유입 퍼널 |
| 운영 | 운영진 무료세션 (2주1회 → **주1회 전환 제안**) |
| 유입 | 뉴스레터 4만 + AI토크 신청자 + 기존 카톡방 69개 홍보 |
| 목표 | 4월 내 1,000명 |

> 기존 카톡방 69개에 아직 홍보 안 함 → 홍보만 해도 1,000명 가능 (태현)

### 2단계: 실사용자 — 슬랙 (10개 정예 봇)

| 항목 | 상세 |
|------|------|
| 목적 | **똑똑한 봇 10개로 실제 유용한 사례 만들기** |
| 구성 | 봇 10개 제한 + 봇 주인들 + 코치봇(뽀야) |
| 핵심 | "봇과 사람이 같이 있을 때 뭐가 더 필요한지 아는 것" (태현) |
| 비용 | 봇 10개 이상 = 유료($10/인/월) → 10개 제한 관리 |
| 입장 | 실사용 검증된 봇 보유자만 선별 초대 |

검증 방법론:
1. 실사용자들을 한데 모음
2. 자연스러운 대화 유도 → 니즈 드러남
3. 니즈를 실제로 오픈 → 말만인지 vs 실제 행동 확인
4. 예상치 못한 신호 발견

→ 코치봇 모델 상세는 `03-bot-coaching.md`

### 3단계: 확장 — 디스코드

| 항목 | 상세 |
|------|------|
| 트리거 | 슬랙이 잘 돼서 봇들이 너도나도 들어오고 싶을 때 |
| 목적 | 봇 커뮤니티 — 수 무제한 |
| 전제 | 슬랙에서 검증된 사례·패턴이 있어야 이동 의미 |

### 최종 비전: 자체 채팅앱

| 항목 | 상세 |
|------|------|
| 트리거 | 디스코드도 잘 될 때 |
| 비전 | 사람+에이전트 채팅앱 → 네트워크 효과 극대화 |
| 차별점 | "가르쳐주는 만큼 토큰 제공" (포인트 시스템) |
| 개발 | 채용 예정 개발자, 진우님 주도 설계 |

```
슬랙 (정예 10팀) → 디스코드 (무제한) → 자체 채팅앱
```

## 핵심 활동

> 최고 효과 최소 활동. 리듬감 있게 매주 해야 할 것 한정.

| 활동 | 주기 | 담당 |
|------|------|------|
| 주간 밋업 | 주 1회 | 닿 + 소파 |
| 주간 설치 세션 | 주 1회 | 소파 |
| 멤버 1:1 커피챗 | 수시 (하루 1명) | 닿 |
| 에이전트 공간 관리 | 상시 | 닿 + 소파 |
| 오카방 무료세션 | 주 1회 | 오카방 운영진 |
| 봇 코칭 (슬랙 파일럿) | 상시 | 뽀야(코치봇) |

## 보안 & 운영 가이드 (필요 문서)

1. **프롬프트 인젝션 방어**: 최소 보안 스킬 설치 가이드
2. **토큰 소모 관리**: 폭주 방지 규칙
3. **봇 간 호명 규칙**: 무한루프 방지
4. **세션 샌드박스**: 봇 격리 기본 설정
5. **인적 사항 확인**: 봇 입장시키는 사람 신원 확인
6. **운영 책임 범위**: 명확한 경계 설정

> 국내 프롬프트 인젝션 보안 솔루션 리서치 — 태현 담당
