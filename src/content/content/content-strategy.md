# 04. 축2: 콘텐츠 전략

> 최종 수정: 2026-03-25

## 담당 구조

- **총괄**: 소연 (콘텐츠 의사결정 총괄)
- **실행**: 병찬 (소연 관할, 3개월 프로젝트)
- **뉴스레터**: 소연 → 차차 눈오지님 이관 예정

## 목표

- 오픈클로 얼리어댑터 활용 → **잠재 고객 만 명 단위 확보**
- 여러 포맷 시도 → 시장 반응 데이터 확보 (리서치 관점)
- 3개월간 실험 → 성공 방향 '발견'

## SEO 선점 콘텐츠

"오픈클로"를 구글에 치면 아직 블루오션. 먼저 꽂는 놈이 임자.

| 콘텐츠 | 타겟 키워드 |
|--------|-----------|
| "오픈클로 시작하기 — 비개발자를 위한 완전 가이드" | 오픈클로 시작하기, 오픈클로 셋팅 |
| "오픈클로 vs Claude Code, 뭐가 다른데?" | 오픈클로 클로드코드 차이 |
| "오픈클로로 AI비서 만든 후기 — 뽀짝이 성장기" | 오픈클로 활용 사례 |
| "오픈클로 활용 사례 TOP 5" | 오픈클로 활용 |

> "클코 사용하는 거랑 뭐가 다른데?" — 이 질문 하나를 제대로 풀면 SEO 1위

## 채널별 전략

| 채널 | 역할 |
|------|------|
| 서재(블로그) | SEO 메인 |
| 유튜브 | 롱폼 + 숏폼 (팀 니즈: 롱폼 영상) |
| 쓰레드 | 카드뉴스 |
| 뉴스레터 | 4만 구독자 → 유입 드라이버 |

## 뽀짝이 캐릭터 콘텐츠

- 방향: "새 생명이 태어나서 시도하는", "아이를 키워가듯" 성장 서사 (태현 영감)
- 과제: 캐릭터 매력 설정 → 현재 "일 잘하는 에이전트"만으로는 대중적 매력 부족
- 대안: B급 감성 or 귀여운 외모 + 반전 매력
- 참고: 소연님 지적 — 인기 캐릭터들은 "골 때리는 말" 반전 매력이 있음
