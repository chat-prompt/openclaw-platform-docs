## 💰 비즈니스 모델

### 핵심 가치 제안

> "코딩 모르는 비개발자도, 결제 후 10분 안에 슬랙에서 AI Agent가 일을 시작합니다."

- **셋업 대행**: OpenClaw 셋업의 진입장벽(Ubuntu, VPS, API 키 등)을 대신 해결
- **프리셋 에이전트**: 현업 Knowledge가 사전 탑재된 커스텀 에이전트 제공
- **1회 구매 → 평생 소유**: 월구독이 아닌 일시불 모델 (구독피로 공략)

### 가격 체계 (슈퍼 얼리버드 55% 할인, ~3/27 마감)

| 플랜 | 정가 | 할인가 | 포함 내용 |
|------|------|--------|----------|
| **Research Assistant Agent** | ~~131,000원~~ | **59,000원** | RA 봇 1종 세팅 + 1만원 AI 토큰 + 활용 가이드 + 웨비나 + 커뮤니티 |
| **AI Super Team** ⭐ 인기 | ~~553,000원~~ | **249,000원** | 6종 봇(Research + Secretary + Data + Marketer + Sales + Docs) + 5만원 토큰 + 웨비나 + 커뮤니티 |

> ※ 기본 제공 AI 크레딧 소진 후에는 본인의 API 키를 활용해야 함 (안내 포함)

### 3대 독점 혜택 (자칭)

1. **Slack 팀에 세팅하는 데까지 3분** — "일일이 가이드 보고 연동하는 데 2~3시간 이상 걸려요. 그 소요를 단 3분만에 딸깍으로 해결"
2. **현업 Knowledge가 탑재된 봇들** — "0부터 시작하면 기본 구성과 역할부터 제대로 못 잡을 수 있어요. 토스 PO, 크래프톤 AI Lead, 서울대 마케터의 Knowledge 기본 탑재"
3. **슈퍼 얼리버드 한정 혜택** — 55% 할인 + 무료 토큰 + Closed 웨비나 + Closed 커뮤니티

## 🧩 Deskmate SWOT 분석

### 💪 Strengths (강점)
- 셋업 진입장벽 해소 — **가장 큰 페인포인트 직격**
- "3분 세팅" 명확한 후크
- 1회 구매 모델 → 구독피로 우회
- 빠른 실행력 (런칭부터 웨비나까지 단기)

### ⚠️ Weaknesses (약점)
- 커뮤니티 42명 — 아직 매우 초기
- "토스/크래프톤 출신" 검증 불가 (팀원 이름 비공개)
- 프리셋 봇 = 범용적 → **실제 커스텀 한계**
- 1회 구매 후 지속 수익 모델 불투명
- OpenClaw 공식이 아닌 서드파티 → 업데이트 추종 리스크

### 🔭 Opportunities (기회)
- OpenClaw 셋업 수요 급증 (초보자 유입↑)
- 오프라인 밋업 → 충성 커뮤니티 빌딩 가능
- "세팅 대행" → "운영 컨설팅" 확장 여지

### 🔥 Threats (위협)
- **GPTers(우리)**: 압도적 커뮤니티 + 콘텐츠 + 기술 우위
- OpenClaw 자체 온보딩 개선 시 존재이유 희석
- "셋업 대행"만으로는 해자(moat)가 약함
- 신뢰 리스크 — 팀 신원 비공개

## 💡 시사점 & 대응 방향

### 🎯 핵심 인사이트

> Deskmate는 **"셋업의 고통"이라는 진짜 페인포인트**를 정확히 짚었다. 우리도 이 니즈를 인지하고 있으나(타타 제안: 세팅 모각, 세팅 웨비나), 아직 체계적으로 대응하지 못했다. **셋업 지원을 유료 서비스가 아닌 커뮤니티 가치로 제공**하면 Deskmate의 유료 모델을 무력화할 수 있다.

### ⚠️ 위협 요소
- **"오픈클로 코리아"라는 네이밍** → 공식 한국 채널로 오인 가능 (브랜드 리스크)
- 초보자 대상 "설치해주기" 니즈를 먼저 선점하면 유입 빼앗길 수 있음
- 3/27 얼리버드 마감 + 오프라인 밋업 → 빠른 전환 시도 중

### ✅ 기회 요소
- 셋업 고통 = **우리 콘텐츠/이벤트 소재** (세팅 모각, 무료 웨비나)
- 프리셋 봇의 한계 → "진짜 커스텀은 스킬을 직접 만드는 것" 교육 방향
- 우리의 규모(4만 뉴스레터, 465명 스터디) 대비 42명은 미약 → **시간 여유 있음**
- "오픈클로를 검색하면 GPTers"가 나오는 콘텐츠 전략 계속 밀기
