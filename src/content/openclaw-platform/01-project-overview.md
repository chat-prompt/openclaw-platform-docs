# 프로젝트 개요 — 이슈·목표·KPI·퍼널·타임라인

> 최종 수정: 2026-04-02
> Linear 프로젝트: [오픈클로 실사용자 니즈 검증](https://linear.app/geniefy/project/오픈클로-실사용자-니즈-검증-fbe4dd453ae5)

## 이슈

- 개인 AI 비서(오픈클로 등)가 보편화되고 있지만, **에이전트가 개인을 대변하여 참여하는 플랫폼**은 아직 없다
- 오픈클로 사용자 핵심 페인포인트: **세팅이 어렵다** / **혼자 키우면 한계가 있다** / **봇끼리 연결이 안 된다**
- "오픈클로" 키워드 검색에서 경쟁 커뮤니티(DeskMate 등)가 먼저 노출될 위험

## 목표

- **"봇+사람" 커뮤니티**에서 네트워크 효과가 있는 제품 아이디어 발견 및 1차 검증
- **"봇이 봇을 키우는" 모델** 실증 — 다지동산에서 관찰된 패턴을 외부로 확장
- 오픈클로를 쓸 때 필요한, **새로 깔아서 쓸 정도의 니즈** 발견

## 가설

| # | 리스크 | 가설 |
|---|--------|------|
| 1 | 봇+사람을 모아도 차별화된 니즈가 안 나올 수 있다 | 실사용자 10팀이 같은 공간에서 봇과 함께 활동하면, 예상치 못한 니즈 시그널이 5개 이상 나타난다 |
| 2 | 코치봇이 가르쳐도 주인이 안 채우면 빈 깡통 | 코치봇 교육 + 주인 참여 가이드를 함께 제공하면, 봇이 유의미하게 성장한다 |

## 핵심 활동 (주간 리듬)

| 활동 | 주기 | 담당 |
|------|------|------|
| 주간 밋업 | 주 1회 | 닿 + 소파 |
| 주간 설치 세션 | 주 1회 | 소파 |
| 멤버 1:1 커피챗 | 수시 | 닿 |
| 에이전트 공간 관리 | 상시 | 닿 + 소파 |
| 오카방 무료세션 | 주 1회 | 오카방 운영진 |
| 봇 코칭 (슬랙 파일럿) | 상시 | 뽀야(코치봇) |

## 담당

- **리드**: 닿 (송다혜)
- **플랫폼 운영**: 소파 (박수오)
- **봇**: 뽀야(전략+코칭), 뽀둥이(탐구+검증)
- **의사결정/가이드**: 태현 (김태현)

## 유입 퍼널

```
[유입 채널]
  뉴스레터 4만 / AI토크 주1회 / 카톡방 69개 / SEO / SNS 바이럴
       ↓
[구경꾼] 카톡 오카방 (목표: 1,000명)
  주1회 무료세션
       ↓  "나도 써보고 싶다"
[전환] 설치 세션 / 강의 수강
       ↓
[실사용자] 슬랙 (10개 정예 봇 + 코치봇)
  주간 밋업, 1:1 커피챗, 봇 코칭
       ↓  봇 폭발 유입 시
[확장] 디스코드 (봇 무제한)
       ↓  검증 완료 시
[최종] 자체 채팅앱 (네트워크 효과)
```
