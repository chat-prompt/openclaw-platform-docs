## 네이밍 체계

### 유저 성장 4단계 (커뮤니티)

| 단계 | 이름 | 대상 |
|------|------|------|
| 0 | **Starter** | 미설치 / 설치만 |
| 1 | **Crew** | 봇 1마리 활용 |
| 2 | **Captain** | 2마리+ 협업 |
| 3 | **Commander** | 조직단위 다머신 |

→ 최종 목표: Commander를 최대한 많이 만드는 것
→ 육성 + 실험 + PMF 검증

### 밋업 티켓 2종 (별도)

| 티켓 | 이름 | 대상 | 역할 |
|------|------|------|------|
| 상위 | **Maker (메이커)** | 봇 2마리+ 협업 경험 | 콘테스트 참가 + 소그룹 리드 + 수익화 실험 기회 |
| 일반 | **Viewer (뷰어)** | 관심 있는 누구나 | 세션 참여 + 네트워킹 + 투표 |

### 유저 성장 퍼널

```
Starter (미설치/설치만)
  ↓ 웨비나 시리즈 (4/3, 4/10, 4/14)
Crew (봇 1마리)
  ↓ 뽀야 질문방 (벽 돌파 코칭)
Captain (2마리+ 협업)
  ↓ 4/25 밋업 Maker → 콘테스트 → 수익화 실험
Commander (조직단위 다머신) ← 최종 목표
  ↑ 수익화 실험 결과 기반으로 경로 설계
```

### 바이럴 플라이휠

```
웨비나 → 후기 → SNS → 덕후방 유입 → 다음 웨비나
                                         ↓
                               질문방 → 밋업 참가
                                         ↓
                               밋업 → 현장 인증 → SNS
                                         ↓
                               → 덕후방 유입 → 다음 프로그램
```

### Viewer→Maker 승급 챌린지 — 2안

**A안: 현장 승급**
- 조건: 밋업 당일 현장에서 봇 2마리 협업 세팅 성공
- 인증: 운영팀 또는 뽀피터스 봇에게 시연 확인
- 혜택: 즉시 Maker 자격 + 다음 밋업 Maker 슈퍼얼리버드
- 특징: 드라마틱한 순간 = 바이럴 소재. 성공자 극소수 예상

**B안: 2주 후 승급**
- 조건: 밋업 후 2주 내 (5/9까지) 봇 2마리 협업 인증
- 인증: 에어테이블 폼에 협업 로그/스크린샷 제출
- 혜택: 다음 밋업 Maker 자격 + 슈퍼얼리버드
- 특징: 현실적으로 달성 가능 → Maker 풀 확대

→ A+B 병행도 가능 (현장 성공 = 즉시 승급, 못 해도 2주 내 도전 가능)
