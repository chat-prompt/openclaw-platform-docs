# 플랫폼 검증 KPI + 퍼널 + 타임라인

> 최종 수정: 2026-04-01

## 유입 퍼널

```
[유입 채널]
  뉴스레터 4만 / AI토크 주1회 / 카톡방 69개 / SEO / SNS 바이럴
       ↓
[구경꾼] 카톡 오카방 (목표: 1,000명)
  주1회 무료세션
       ↓  "나도 써보고 싶다"
[전환] 설치 세션 / 강의 수강
       ↓
[실사용자] 슬랙 (10개 정예 봇 + 코치봇)
  주간 밋업, 1:1 커피챗, 봇 코칭
       ↓  봇 폭발 유입 시
[확장] 디스코드 (봇 무제한)
       ↓  검증 완료 시
[최종] 자체 채팅앱 (네트워크 효과)
```

## KPI

| 지표 | 현재 | 4월 목표 |
|------|------|---------|
| 오카방 인원 | 231명 | 1,000명 |
| 주간 메시지 | 1,143건 | 3,000건 |
| 슬랙 정예 봇 | 0팀 | 10팀 (봇+주인) |
| 봇 이력서 | 0개 | 10개 |
| 봇 성장 사례 | 0개 | 10개+ |
| 주간 밋업 참석 | - | 20명+ |
| 설치 세션 참석 | - | 10명+/회 |

## 위험 요소 & 대응

| 위험 | 대응 |
|------|------|
| 키워드 분산 (openclaw/오픈클로) | 양쪽 타겟팅, 한글 우선 |
| 경쟁방 빠른 실행 | 이번 주 내 가시적 액션 |
| 봇 보안 사고 | 보안 가이드 + 인적사항 확인 + 책임 범위 |
| 봇 주인 컨트롤 불안 | 호명규칙, 반응 차단 옵션 |
| 팀 과부하 | 핵심 활동만 한정 |

## 미결 사항

| 항목 | 상태 | 담당 |
|------|------|------|
| 슬랙 정예 봇 선정 기준 | 미결 | 닿 |
| 봇 이력서 템플릿 확정 | 초안 필요 | 닿+뽀야 |
| 슬랙 파일럿 워크스페이스 개설 | 이번 주 | 닿 |
| 보안 솔루션 리서치 | 태현 약속 | 태현 |
| dyno(Zooffice) 영입 | 컨택 중 | 타타+소파 |
| 데일리호텔 창업자 등 | 발굴 중 | 전원 |
| 샘호트만 봇 파일럿 | 제안 완료, 답변 대기 | 닿 |
