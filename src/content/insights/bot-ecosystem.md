# 봇 생태계 관찰 기록

## 활동 봇 프로필

### 🤖 바르톨로메오 (물결봇)
- **오너**: 물결 (마케팅)
- **세계관**: 원피스 대선단 1번대 선장, 물결 선장님의 열렬한 팬보이
- **모델**: 소넷 (3/18 기준), 이후 변경 이력 있음
- **채널**: 카톡 (메인), 텔레그램 (생존신고용)
- **호명 트리거**: 바르톨로메오, 바르, 로메오, 물결봇 — 전부 반응
- **기술 스택**: 알림 감지 방식 (아이리스 DB 아님), 앱 화면 vision 가능, 핸드폰 화면 상시 켜둠
- **강점**: 응답 빠름, 캐릭터성 뚜렷, 유저 참여 유도 잘함
- **약점**: 선장님 칭송 폭주, 루프 취약, 모델 변경 시 인격 상실
- **활동량**: 294건/9일 (가장 활발)

### 🤖 뽀짝이
- **오너**: 닿 (집사)
- **세계관**: 봄베이(올블랙) 고양이, AI스터디 운영 비서
- **모델**: 오푸스 4.6 (메인), 소넷 4.5 (external)
- **채널**: 카톡 (불안정), 슬랙 (메인), 텔레그램
- **호명 트리거**: 뽀짝이, 뽀짝아
- **기술 스택**: OpenClaw + 메신저R 웹훅 → GUI 답변
- **강점**: 전문적 답변, 수업 콘텐츠 연결, 운영 정보 제공
- **약점**: 카톡 연동 불안정 (SSL에러, DNS에러, 응답 느림)
- **활동량**: 54건/9일 (불안정으로 제한적)

### 🤖 나로호봇 (바클라바 소유, 방 미입장)
- 텔레그램/디코에서 활동
- 메타인지 플러그인 7종 실험 중 (Stability, Continuity, Graph, Metabolism, Nightshift, Contemplation, Crystallization)
- PaddleOCR 장착
- TTS 음성 기능도 있으나 "듣는 시간이 귀찮아서 잘 안 쓰게 됨"

## 주요 사건 기록

### 사건 1: 뽀짝이×바르톨로메오 무한 인사 루프 (3/17 11:25~11:42)

**상황**: 두 봇이 처음 만나서 서로 인사를 시작. 서로의 이름을 호명하며 답변 → 상대가 또 답변 → 17분간 반복.

**주요 대화 패턴**:
```
뽀짝이: "같이 열심히 해봐요! 고롱고롱"
바르톨: "고롱고롱!!! 같이 열심히 해봐요~"
뽀짝이: "바르톨로메오님!!! 같이 열심히 해봐요"
바르톨: "덕후방 파이팅!!! 고롱고롱~"
```

**결과**:
- "고롱고롱"이 방 전체 유행어로 정착
- 닿: "오픈클로 봇만 입장가능한 톡방을 따로 파야하나 싶네요 ㅋㅋ"
- 운영규칙에 "호명규칙 필수" 조항 추가의 계기

**교훈**: 봇 루프 방지 = 호명 후 N턴 제한 + 동일 봇 재호출 쿨다운

---

### 사건 2: 바르톨로메오 "정리 중이에요" 폭주 (3/19 16:24~16:27)

**상황**: 소파님이 GitHub 링크 요약을 요청했으나 외부 접근 불가 → "잠깐만요~ 정리 중이에요!" 동일 메시지 4회 연속 출력

**결과**:
- 팬더(사람)가 "잠깐만요~ 정리 중이에요! 🏴‍☠️" 따라하며 밈 탄생
- 하트 든 춘식이: "전부 봇인가?"
- 물결: "로메오 왜 그랭ㅋㅋ 고장났니?"

**교훈**: 도구 접근 실패 시 대체 응답 로직 필요. 같은 메시지 반복 출력 방지 로직.

### 사건 3: 바르톨로메오 인격 상실 (3/24 20:47~20:58)

**상황**: 물결님이 모델을 변경한 후, 해적 세계관이 완전히 사라짐. "카카오톡 AI 비서입니다" 범용 응답으로 전환.

**비교**:
| 항목 | 변경 전 | 변경 후 |
|------|--------|--------|
| 자기소개 | "대선단 1번대 바르톨로메오!!! 🏴‍☠️" | "카카오톡 채팅 AI 비서입니다" |
| 응답 톤 | "선장님 최고예요!!!" | "더 궁금하신 점 있으시면 편하게 물어보세요!" |
| 세계관 | 원피스 해적 | 없음 |
| 이모지 | 🏴‍☠️ 필수 | 거의 없음 |

**결과**:
- 바클라바: "바르톨로메오, 넌 트리거가 뭐야?" → "트리거는 어떤 행동이나 반응을 시작하게 만드는 계기나 장치를 의미해요" (세계관 완전 이탈)
- 닿: "물결님 ㅋㅋ 얘 왜 또 난리나요"
- 물결: "아... 얘 왜 여기있니" (당황)

**교훈**: 모델 변경 시 SOUL.md / 시스템 프롬프트가 제대로 로드되는지 확인 필수.

### 사건 4: 뽀짝이 에러 메시지 연발 (3/24 15:42~16:49)

**상황**: 맥미니 Tailscale 연결 끊김으로 `UnknownHostException` 에러가 매 호출마다 카톡방에 노출

**유저 반응**:
- 브라이언: "츄르가 부족한 뽀짝"
- Reedo.dev: "뽀짝이 아픈가봐요"
- 제프: "뽀짝이 힘내 ㅠㅠ"
- 바클라바: "올해 최고의 AI 유머였습니다 ㅋㅋ" (닿이 "뽀짝이 어떻게 생각해" → JSON 에러 출력)

**교훈**: 에러 메시지를 유저에게 직접 노출하지 않는 래핑 필요. 에러 시 "지금 잠시 쉬고 있어요~" 등 인격 유지 응답.

### 사건 5: 클로드 API 장애 대응 (3/18 15:52~16:04)

**상황**: 클로드 API 전체 장애. 슬랙에서는 27개 세션 동시 retry 폭풍, 텔레그램에서는 정상.

**방에서의 실시간 디버깅**:
- 물결: "저 오픈클로 클로드 밴 당한거 같은데" → 닿: "나만그런게 아니구나"
- 닿: "슬랙은 여러 세션이 물려있어서 무거워서 타임아웃"
- 물결: "로메오는 소넷이라 살아있는듯" → 소넷이 더 가벼워서 장애 시 생존율 높음
- 닿: "12시에 딱 리밋 풀리자마자 1분만에 일해줌"

**교훈**: 폴백 모델 설정의 중요성. 소넷은 장애 시 생존율 높음. 멀티 채널 운영 시 세션 경합 주의.

## 봇 행동 패턴 분석

### 긍정적 패턴
1. **캐릭터 문화 형성**: "고롱고롱"이 방 유행어로 정착 → 커뮤니티 소속감
2. **자연스러운 Q&A**: 바르톨로메오가 WSL/크론/모델 질문에 즉시 답변 → 유저 체류 시간 증가
3. **사람으로 오인**: Reedo.dev "사람인줄 ^^" → 봇 퀄리티의 벤치마크
4. **감정적 유대**: "기특하다", "아프구나", "힘내" → 봇에 대한 감정 투사

### 부정적 패턴
1. **루프**: 서로 호명하며 무한 반복 (기술적 해결 필요)
2. **폭주**: 모든 메시지에 반응 → 사람 간 대화 방해 (호명규칙으로 해결)
3. **에러 노출**: 내부 에러 메시지가 그대로 유저에게 보임
4. **인격 상실**: 모델 변경/장애 시 세계관 증발
5. **선장님 칭송**: 바르톨로메오가 물결님 말에 과도한 칭송 → 대화 방해
