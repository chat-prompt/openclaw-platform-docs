### 카톡 덕후방 연동 전략

```
[카톡 덕후방] ←→ [SNS 계정] ←→ [봇키우기교실/서재]
     ↑                ↑                    ↑
   대화·질문      콘텐츠·유입          교육·심화
```

| 연동 포인트 | 구체적 방법 |
|-----------|-----------|
| **카톡 → SNS** | 카톡방 핫토픽을 스레드에 "오늘 덕후방에서 이런 얘기 나왔어요" 포스팅 |
| **SNS → 카톡** | 인스타 바이오/스토리에 카톡방 입장 링크 상시 노출 |
| **카톡 → 교육** | 반복 질문을 봇키우기교실 강의로 연결 ("이건 3강에서 다뤘어요") |
| **교육 → SNS** | 봇키우기교실 수강 인증 → SNS 공유 이벤트 |
| **SNS → 웨비나** | 웨비나 사전 홍보 + 사후 하이라이트 → 다음 웨비나 등록 유도 |

> 핵심: **카톡방은 "깊은 대화", SNS는 "넓은 도달"**. 각자의 역할이 다르므로 콘텐츠를 복사하는 게 아니라 **변환**해서 교차 활용.

### 커뮤니티 → SNS → 커뮤니티 플라이휠

```
      ┌──────────────────────────────┐
      │                              │
      ▼                              │
  [카톡 덕후방]                        │
  959명 → 질문/토론                    │
      │                              │
      │ "이번 주 핫토픽"               │
      ▼                              │
  [SNS 콘텐츠]                        │
  카루셀/릴스/스레드                    │
      │                              │
      │ 도달 & 바이럴                  │
      ▼                              │
  [신규 유입]                          │
  팔로워 → 프로필 링크                  │
      │                              │
      │ "더 깊이 배우고 싶다"           │
      ▼                              │
  [봇키우기교실 / 웨비나]               │
  교육 → 실습 → 성공 경험               │
      │                              │
      │ "같이 배우고 싶다"              │
      └──────────────────────────────┘
           (카톡방 재유입)
```

**플라이휠의 핵심 연료:**
1. **카톡방 대화 → SNS 콘텐츠**: 자체 콘텐츠 소스가 무한 (매일 수십 건의 대화)
2. **SNS 바이럴 → 카톡방 유입**: 카톡방 링크를 SNS 모든 접점에 배치
3. **카톡방 → 교육**: 반복 질문 → 교육 콘텐츠화 → 봇키우기교실 강의 추가
4. **교육 수료 → SNS 인증**: 수료 인증 문화 형성 → UGC → 바이럴

> 이 플라이휠이 돌기 시작하면 **콘텐츠 제작 비용이 점점 줄어드는** 구조. UGC가 늘어나니까.

## 7. 수익화 전략

### 단기 (1~3개월): 무료 → 유료 전환 테스트

| 수익원 | 모델 | 예상 수익 |
|--------|------|----------|
| **봇키우기교실** | 기본 11강 무료 유지 + **심화 3강 유료** (3만원) | 30명 × 3만원 = 90만원/기수 |
| **유료 워크숍** | "나만의 AI비서 만들기" 3시간 (5만원) | 20명 × 5만원 = 100만원/회 |
| **웨비나 유료 전환** | 기본 무료 유지 + **녹화본+자료 패키지** (1만원) | 100명 × 1만원 = 100만원/회 |

> 근거: 설문 니즈 "나 혼자 처음부터 따라하는 가이드" + "내 업종에 맞는 예시 템플릿" → **구조화된 가이드에 지불 의향** 확인.

### 중기 (4~6개월): 스폰서 + 가이드

| 수익원 | 모델 | 예상 수익 |
|--------|------|----------|
| **스폰서드 콘텐츠** | AI 도구 업체 리뷰/소개 (50~100만원/건) | 월 1~2건 = 100~200만원 |
| **유료 가이드북** | "업종별 오픈클로 세팅 가이드" PDF (2만원) | 50부 × 2만원 = 100만원/타이틀 |
| **기업 교육** | B2B "사내 AX 워크숍" (인당 10만원 또는 회당 300만원) | 분기 1회 = 300만원 |
| **제휴 수수료** | 맥미니/클로드 구독 제휴 링크 | 월 10~30만원 |

### 장기 (7~12개월): 체계적 수익 구조

| 수익원 | 모델 | 예상 월 수익 |
|--------|------|------------|
| **정기 교육 프로그램** | "오픈클로 마스터 코스" (10만원/월 구독) | 50명 × 10만원 = 500만원 |
| **기업 컨설팅** | AX 도입 컨설팅 (500만원/건) | 월 1건 = 500만원 |
| **커뮤니티 프리미엄** | 프리미엄 카톡방 or 디스코드 (2만원/월) | 100명 × 2만원 = 200만원 |
| **스킬/템플릿 마켓** | 사용자 제작 스킬 판매 (수수료 30%) | 성장 후 측정 |
| **굿즈/IP** | 뽀야/뽀짝이 스티커팩, 굿즈 | 부수입 |

**수익화 핵심 원칙:**
1. **무료 콘텐츠가 깔때기 입구** — 봇키우기교실 기본은 영원히 무료
2. **유료는 "시간 절약"에 과금** — 정보는 무료, 정리된 가이드/개인 피드백은 유료
3. **B2B가 진짜 수익** — SNS는 B2B 리드 생성 퍼널. 기업 교육이 고마진

### 🔥 액션 5: 카톡방 "주간 핫토픽" 정기 코너 시작 (다음 주 금요일부터)

- 스레드에 "이번 주 덕후방 다이제스트" 포스팅 가능
